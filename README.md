What the plugin does?

You connect a NVIDIA NIM Ai model to the plugin, and it works!
What it does:
- Checks for types according to action :v:
- Automaticly adds types if AutoCorrect is on :Roblox: 
- Has bindable Shortcut to Ask the AI to update a line!
- Works great when Speed Typing a prototype!
- Free... Until i make a paid plus version with better features!

How to get an API key?

you go here [Nvidia Models](https://build.nvidia.com/models)
Look for a free endpoint NVIDIA model - Tested: ( Nvidia/nemotron-3-nano-30b-a3b and Nvidia/nemotron-3-ultra-550b-a55b )
Get an API key for the model and insert the api key and the model name in the required spaces. from now on it will automatically work for you every time you start coding!
